# Dynamic Resume Builder

Dynamic Resume Builder is a web-based application that allows users to create customizable and professional resumes with ease. The application includes dynamic sections for personal information, work experience, education, skills, social links, and more. It supports live preview and print-ready PDF generation.


![resume-template-version2 vercel app_ (1)](https://github.com/user-attachments/assets/3191b246-2eb9-4d05-8a29-3cd87c575a13)


## Features
- **Dynamic Sections**: Add, edit, and remove sections like work experience, education, skills, and certifications.
- **Live Preview**: Real-time preview of your resume while editing.
- **Social Links**: Add links to LinkedIn, GitHub, Twitter, Instagram, and other platforms.
- **Custom Styles**: Print-ready, colorful, and professional resume styles.
- **Responsive Design**: Works on both desktop and mobile devices.
